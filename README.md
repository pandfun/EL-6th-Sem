## Setup

- Create a Virtual Env (optional)
```bash
python3 -m venv myenv
source myenv/bin/activate
```

NOTE : Check the activiation for your OS

---
<br>


### Install Dependencies
```bash
pip install -r requirements.txt
```

### Running the App

- First run the `model.py`
```bash
python3 model.py
```

- Run the `app.py` file
```bash
python3 app.py
```
